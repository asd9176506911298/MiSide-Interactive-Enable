# InteractiveEnable

## Installation / 安裝
- Unzip the contents of the archive into the game folder
- 解壓縮檔案到遊戲資料夾

## Custom Keybinds / 自訂按鍵
- Keybinds can be customized in:
BepInEx\config\InteractiveEnable.cfg
- 可以修改 BepInEx\config\InteractiveEnable.cfg 裡面的按鍵

## Game / 遊戲
- Replay TV Game Fight Penguin Need to Toggle InteractiveEnable and TV/Dance Game Infinite
- You can change music when music playing but suggest dance music end and change music is better
- 重玩 奶盒 企鵝 需要開啟 InteractiveEnable 和 TV/Dance Game Infinite
- 播放音樂時可以更換音樂，但建議音樂結束後更換音樂更好

## Features / 功能
- F3 Show Menu / F3 顯示選單
- Activate Interactive Objects / 啟用物件
- Replay TV Games and customize Fight Enemy Difficulty. / 重玩奶盒企鵝遊戲 調整奶盒敵人強度
- Open Door / 開門
- Dance Game Replay Change Music / 重玩跳舞遊戲 換歌
- Teleport to Game / 傳送至遊戲前
- FPS Game Invincible One Hit Kill / FPS遊戲 無敵 一級必殺
